# Sujet NextJS 5IW

## Contexte

Ce sujet permet de rajouter des instructions pour le réalisation du
front pour le projet Clean Architecture. Ce front sera évalué et
servira de note de partiel pour la matière NextJS.

## Instructions

- Votre application doit respecter une approche atomic design de
  ses composants
- Utilisation de contexte(s) pour partager des states entre
  composants
- Gestion des formulaires avec React Hook Form et validation
  avec des schémas (exemple: zod)
- Les pages 404 et 500 doivent être correctement intégrées sur
  votre site en accord avec la charte graphique de votre
  application
- Votre application doit être traduite en français et en anglais
- Un sitemap.xml doit être accessible pour lister les pages de
  votre application
- Au moins une page d’accueil avec les metadata correctement
  intégré pour le SEO
- Votre application doit avoir un maximum de rendu côté serveur
- Votre application doit être conçue de sorte à utiliser un
  maximum de cache, qu’il soit applicatif ou API

## Bonus

- Cache géré par un Redis
- Animations de tableaux, cards, listes, ...
- Drag’n’Drop (exemple: pour déplacer de l’argent d’un compte à
  un autre)
